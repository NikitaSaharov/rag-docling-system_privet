# Тестовый документ о Docker

Docker - это платформа для контейнеризации приложений.

## Основные компоненты

- **Docker Engine** - ядро системы
- **Docker Hub** - репозиторий образов
- **Docker Compose** - инструмент оркестрации

## Преимущества

1. Изоляция приложений
2. Портативность
3. Быстрое развертывание

Docker позволяет упаковать приложение со всеми зависимостями в контейнер.
